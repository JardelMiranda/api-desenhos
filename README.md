# Projeto API Front-End

Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento Web Front-End.

## 📌 Descrição
A aplicação consome dados de uma API pública e exibe personagens em forma de cards dinâmicos na tela.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 🔗 API utilizada
Rick and Morty API  
https://rickandmortyapi.com/

## ⚙️ Funcionalidades
- Consumo de API com fetch
- Exibição dinâmica dos personagens
- Busca em tempo real

## ▶️ Como executar
1. Baixe os arquivos
2. Abra o arquivo `index.html` no navegador
